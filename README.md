## 🛠️ AWS Services Used

- Amazon S3 – Static website hosting  
- Amazon CloudFront – CDN for global delivery  
- Amazon Route 53 – DNS management  
- AWS Certificate Manager (ACM) – HTTPS security  

---

## ⚙️ Deployment Steps

1. Created S3 bucket and enabled static hosting  
2. Uploaded website files (HTML, CSS, images)  
3. Requested SSL certificate using ACM  
4. Configured CloudFront distribution  
5. Connected domain using Route 53  

---

## 🔐 Features

- Custom domain (logesh.online)  
- HTTPS enabled  
- Fast global delivery using CDN  
- Scalable and cost-effective
